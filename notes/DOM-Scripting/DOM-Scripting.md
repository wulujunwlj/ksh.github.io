### 1.3 浏览器战争
* DHTML(Dynamic HTML)：动态HTML
    - DHTML 并不是一项新技术，而是描述HTML、CSS和JavaScript技术组合的术语
    - 利用 HTML 把网页标记为各种元素
    - 利用 CSS 设置元素样式和它们的显示位置
    - 利用 JavaScript 实时地操控页面和改变样式

### 1.4 制定标准
* W3C 推出了一个标准化的 DOM
* DOM 是一种 API
* W3C 对 DOM 的定义：一个与系统平台和编程语言无关的接口，程序和脚本可以通过这个接口动态地访问和修改文档的内容、结构和样式
